## 项目具体信息

本系统基于NODEMCU、GSM芯片，主要通过超声波、温度传感技术作为监测手段，对垃圾桶进行状态分析，然后通过NODEMCU模块将有效数据上传到阿里云IoT开发平台将数据进行可视化处理，从而用户可以轻松有效的获取所需信息。于此同时通过GSM模块向工作人员发送提醒短信，到达高效便捷、用户友好的“采样-计算-互联-处理”机制，助力解决当下市政垃圾回收中存在的重点、疼点、难点问题。
